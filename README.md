Strong gravitational lensing simulations currently require constant manual intervention. Researchers must configure parameters, validate outputs, retrain downstream classifiers and iterate on failures by hand. This bottleneck limits large-scale dataset generation and forces physicists to spend time on engineering tasks rather than scientific discovery.

This repository presents a production-ready Agentic AI framework built on Pydantic AI that autonomously orchestrates the DeepLenseSim / lenstronomy simulation pipeline. My proof-of-concept accepts natural-language requests, uses a human-in-the-loop (HITL) clarification loop to resolve ambiguous parameters, validates all inputs via Pydantic v2 models, executes lenstronomy simulations and returns structured metadata alongside generated images. The GSoC project will extend this foundation into a full multi-agent system capable of SLURM job submission, failure recovery, dataset quality validation, and reinforcement-learning-guided parameter exploration.
